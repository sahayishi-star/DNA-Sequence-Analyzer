# DNA Sequence Analyzer

A beginner Biopython project.

## What it does
- Detects Open Reading Frames (ORFs)
- Transcribes DNA → mRNA
- Translates mRNA → amino acid sequences
- Calculates GC content and molecular stability
- Plots k-mer GC composition graphs
- K-mer sliding windows for thermodynamic stability based on H-bonding
- K-mer sliding windows for thermodynamic stability based on SantaLucia Nearest-Neighbor model
- Salt concentration corrections and pentalites added to SantaLucia model

## Tools used
- Python
- Biopython
- Matplotlib
- Ipywidgets
- Numpy
- Google Colab
